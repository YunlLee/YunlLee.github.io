1. Ada AD 102 GPU

The full AD102 GPU includes 12 Graphics Processing Clusters (GPCs), 72 Texture Processing
Clusters (TPCs), 144 Streaming Multiprocessors (SMs), and a 384-bit memory interface with 12
32-bit memory controllers.

2. GPC

SM(Stream Multiprocessor)

  • PolyMorph Engine: 多边形变形引擎，主要用于处理多边形和顶点相关的工作

  • Core：用于执行各种各样的渲染指令，图形渲染的运算核心，又叫 SP(stream processor)

  • warp scheduler：用于调度 Core，以最大限度的利用渲染核心

  • Instruction Cache：每一个渲染核心执行的指令是一样的。指令都是从指令缓冲中读取，通过 Dispatch Unit 分配到每一个Core 上面。

  • SFU：特殊运算单元。主要用来执行一些高阶的数学运算。如：指数，对数，三角函数等。

  • LD/ST：加载或保存临时数据、寄存器数据的一些模块。

  • Register File：存储数据的缓存区。这里的数据访问速度是最快的。

  • L1 Cache: 数据访问速度慢于 Register File，但容量更大。

  • Uniform Cache：全局统一缓存。

  • Tex unit：纹理采样单元。在 Fermi 机构下，一个 SM 有 4 个纹理采样的单元，每个单元有 4 个通道。我们刚好可以采样 16 个通道的数据。这也是为什么有的显卡会限制你在一个shader里面只能使用16个采样器的原因。

  • Texture Cache：纹理缓存

  • Interconnect Network：链接各个核心之间的网络。帮助核心之间数据的交换。
