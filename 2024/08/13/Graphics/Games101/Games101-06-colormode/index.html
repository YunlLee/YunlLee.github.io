06.Games101--颜色模型

1. 颜色模型

1.1 RGB 颜色空间

  • 色彩使用三通道 RGB 向量(r, g, b)来表示
  • 通常可以将(r, g, b) 分别规整化为[0,1]内的浮点数；当使用8bit存储时，RGB通常取值[0,255]内的整数。
  • 其它的颜色可以根据这三个颜色的线性组合表示：C = rR + gG + bB;
  • 缺点：RGB 并不能表示所有的颜色
  • 称之为加色系统 (0, 0, 0)为黑、 (1, 1, 1)为白

1.2 CMY 颜色空间

  • Cyan(青)、Magenta(品红)、Yellow(黄)
  • C/M/Y = 1 - R/G/B
  • 称为减色系统 (0, 0, 0)为白、 (1, 1, 1)为黑
  • 印刷行业使用

1.3 HSV 颜色空间

  • 提供了一个比 RGB 更加直观的方式来表示颜色
  • 图像处理，光线追踪，分形图像
  • HSV: 圆锥形的的色彩空间
    • Hue(色调)也叫色相，它描述了颜色的基本属性。
    • Saturation(饱和度) 也叫纯度，饱和度越低，色彩越白。
    • Value of brightness(亮度) 亮度越低，色彩越黑。

1.4 CIE XYZ 色彩空间

  • 可以表示所有可感知的颜色

2. 图像和像素

  • 图像可以看成是一个二维离散函数: f(x,y)
  • 函数 f的定义域是由矩阵排列着的许多格了组成，这些格子被称为 像素(pixel)
  • 函数 f的取值则为各个像素的色彩:对于彩色图像，可以是 RGB 或者 RGBA;对于灰度图像，f 为单值函数

3. 三角网格模型

图形学的基本目标是什么?
从虚拟的三维场景及相机的位置信息中，生成出一幅二维图像。
而三维场景又以怎样的数据结构来表示?
简单的球体，长方体可直接用其参数描述。
对于复杂模型，则需要使用参数曲线和曲面或者更一般的网格模型来进行描述。
网格模型之中又以三角网格最为常用。

3.1 三角网格定义

三角网格是由一系列欧式空间中的三维顶点以及连接这些顶点的若干三角面片组成，具体包括:

  • 顶点几何 V(v1, v2, … vn)
  • 面片几何 F(f1, f2, … fn)

面片的法向量：

  • 三角面片的法向量是垂直于该三角面片所在平面的非零向量;
  • 对于每个三角面片单独而言，其法向量都有两种可能的朝向;
  • 法向量的朝向决定了一个三角面片的正面与反面;
  • 对于连续可定向的三角网格整体而言，相邻的三角面片需要具备一致的法向量朝向。

计算法向量：
三角网格顶点的法向量可以通过其周围的所有三角面片的法向量通过加权叠加计算:

  • 假设v 是k 个三角面片 f1f_{1}f1​,f1f_{1}f1​…,fkf_{k}fk​,所共有的顶点;
  • 按算数平均计算:
    • Nv=(Nf1+...+Nfk)/kN_{v} = (N_{f1} + ... + N_{fk})/kNv​=(Nf1​+...+Nfk​)/k
  • 面积加权：
    • Nv=(Sf1Nf1+...+SfkNfk)/(Sf1+...+Sfk)N_{v} = (S_{f1}N_{f1} + ... + S_{fk}N_{fk})/(S_{f1}+...+S_{fk})Nv​=(Sf1​Nf1​+...+Sfk​Nfk​)/(Sf1​+...+Sfk​)
  • 按顶点相邻三角形的角度加权：
    • Nv=(Angf1Nf1+...+AngfkNfk)/(Angf1+...+Angfk)N_{v} = (Ang_{f1}N_{f1} + ... + Ang_{fk}N_{fk})/(Ang_{f1}+...+Ang_{fk})Nv​=(Angf1​Nf1​+...+Angfk​Nfk​)/(Angf1​+...+Angfk​)

网格的简单绘制：

  • 三角网格模型的每个顶点需要指定一个色彩属性。
  • 中间的颜色通过插值计算
  • 指定一个虚拟的光照环境

4. 光照模型

光照模型(lighting model或 illumination model)用于计算光的强度:

  • 局部光照明(LocalLighting)
    • 关注物体直接受到光源影响所产生的光照效果
  • 全局光照明(Global Lighting)
    • 关注阴影效果

5. 光的传播

折射定律：

  • n1∗sinθ1=n2∗sinθ2n_{1}*sin\theta_{1} = n_2*sin\theta_{2}n1​∗sinθ1​=n2​∗sinθ2​
  • 光的传播能量守恒：
    • Li=Ld+Ls+Lt+LvL_{i} = L_d + L_s + L_t + L_vLi​=Ld​+Ls​+Lt​+Lv​
    • LiL_iLi​ 如射光的能量
    • LdL_dLd​ 漫反射光的能量
    • LsL_sLs​ 镜面反射光的能量
    • LtL_tLt​ 折射光的能量
    • LvL_vLv​ 被介质和物体吸收的能量

6. 光的度量

  • 立体角（Solid Angle）
    • 衡量某一物体相对于某一视点 P 的视角大小：dω=ds/r2d\omega = ds / r^2dω=ds/r2
    • 立体角最大为 4π\piπ
  • 光通量（Luminous flux）: 单位时间内通过面元 ds 的光能量，记为 dFdFdF
  • 发光强度（Radiance）：单位立体角内的光通量，记为 I
