3.Games101--Shading

1. Shading 的定义

着色，也称浓淡处理、明暗处理、着色法，是在三维模型或插画中通过不同的亮度表现深度的方法。这是一个对不同物体引用不同材质的这么一个过程。

2. Blinn-Phong 反射模型

Blinn-Phong 模型通过考虑漫反射(diffuse)、镜面反射(specular)和环境光(ambient)来近似材质的光照效果。其中，漫反射光贡献了材质的基本颜色，镜面反射光则提供了高光部分，而环境光则是整体环境的光照效果。

2.1. Shading 是局部的

计算特定着色点反射到 camera 的光，我们需要确定一些数据：

  • viewer direction, v
  • surface normal, n
  • light direction, l

tips: Shadings != Shadow

2.2. 漫反射(Diffuse Reflection)

定义：指光被均匀的散射到每个方向（各向同性），不管我们从什么角度看都是相同的。

Lambert 余弦定理：

单位面积的光与 cosθ=I⋅ncos\theta = I \cdot ncosθ=I⋅n 成比例

光的衰减：

假设近光点的强度(Intensity)为 III, 那么整体的强度即为 4πI4\pi I4πI ，
利用能量守恒：4πI=4πr2I4\pi I = 4\pi r^{2} I4πI=4πr2I
最终我们可以得出远光点光的强度: I/r2I / r^{2}I/r2

Lambertian (Diffuse) Shading:

Ld=Kd(I/r2)max(0,n⋅l)L_{d} = K_{d} (I/r^{2})max(0, n \cdot l) Ld​=Kd​(I/r2)max(0,n⋅l)

LdL_{d}Ld​ 表示反射光
KdK_{d}Kd​ 表示漫反射系数。如果 Kd=1K_{d} = 1Kd​=1 则表示不吸收任何能量，进来多少就出去多少。如果 Kd=0K_{d} = 0Kd​=0，即表示黑色，所有的能量都被吸收。

tips： 漫反射与 v 的方向没有关系

2.3. 镜面光(Specular)

Specular 的强度取决于 v 的方向, v与R(反射光)夹角越小，它们之间的点积就越接近1，高光就越亮。

如果根据 R 和 v 夹角来计算高光会有一个弊端，它的镜面反射会在一些情况下出现问题，特别是物体反光度很低时，会导致大片（粗糙的）高光区域。

可以看到，在镜面高光区域的边缘出现了一道很明显的断层。出现这个问题的原因是观察向量和反射向量间的夹角不能大于90度。如果点积的结果为负数，镜面光分量会变为0.0。你可能会觉得，当光线与视线夹角大于90度时你应该不会接收到任何光才对，所以这不是什么问题。
然而，这种想法仅仅只适用于漫反射分量。当考虑漫反射光的时候，如果法线和光源夹角大于90度，光源会处于被照表面的下方，这个时候光照的漫反射分量的确是为0.0。但是，在考虑镜面高光时，我们测量的角度并不是光源与法线的夹角，而是视线与反射光线向量的夹角。看一下下面这两张图：

右图中，视线与反射方向之间的夹角明显大于90度，这种情况下镜面光分量会变为0.0。这在大多数情况下都不是什么问题，因为观察方向离反射方向都非常远。然而，当物体的反光度非常小时，它产生的镜面高光半径足以让这些相反方向的光线对亮度产生足够大的影响。在这种情况下就不能忽略它们对镜面光分量的贡献了。

解决办法就是引入半程向量(half vector)。我们可以直接计算 n(法向量)和 h(半程向量)的点积去替换 V 与 R 的点击。

h=bisector(v,l)=v+l∥v+l∥h=bisector(v, l) = \frac{v + l}{\left \| v + l \right \| } h=bisector(v,l)=∥v+l∥v+l​

Ls=Ks(I/r2)max(0,cos⁡(α))p=Ks(I/r2)max(0,n⋅h)p\begin{aligned} L_{s} &= K_{s}(I / r^{2})max(0, \cos(\alpha))^{p} \\ &= K_{s}(I / r^{2})max(0, n \cdot h)^{p} \end{aligned}Ls​​=Ks​(I/r2)max(0,cos(α))p=Ks​(I/r2)max(0,n⋅h)p​

为什么会有一个指数P呢？向量之间的夹角确实能够体现两个向量是否足够接近，但是它的容忍度太高，有时候觉得距离已经够远了但是它们之间还是有一个较大的值。如果只是用余弦角生成高光就会看到一个很大的高光，但事实上，真实的高光都集中在很小的区域。通常情况下我们都设置 P 在 [100, 200] 的这个范围内。

高光系数 KsK_{s}Ks​ : 我们通常认为高光为白色，所以我们一般将其设置为 1。
如何去考虑能量的吸收问题？Blinn-Phong 是经验模型这里把它简化了，这里我们主要关注的是能否见到高光。当然，绝对正确的情况下我们是需要考虑这一点的。

2.4. 环境光(Ambient)

环境光是不依赖于任何内容的着色。任何一个点，接受来自环境的光永远都是相同的。它主要的作用是保证没有一个地方是黑色的。

La=KaIaL_{a}=K_{a}I_{a} La​=Ka​Ia​

Blinn_Phong 反射模型就是 Ambient + Specular + Diffuse 的和

L=La+Ld+Ls=KaIa+Kd(I/r2)max(0,n⋅l)+Ks(I/r2)max(0,n⋅h)p\begin{aligned} L &= L_{a} + L_{d} + L_{s} \\ &= K_{a}I_{a} + K_{d}(I/r^{2})max(0, n \cdot l) + K_{s}(I/r^{2})max(0, n \cdot h)^p \end{aligned} L​=La​+Ld​+Ls​=Ka​Ia​+Kd​(I/r2)max(0,n⋅l)+Ks​(I/r2)max(0,n⋅h)p​

3. 着色频率(Shading Frequenices)

3.1. flat shading

为每个三角形面进行着色：

  • 计算面的法线(两条边求向量积)
  • 根据 l、n、v 计算出一个 shading，这个 shading 就是这个三角形的长相。

3.2. Gouraud Shading

为每个顶点着色：
  • 计算法向量(相邻面法向量的平均 Nv=∑iNi∥∑iNi∥N_{v} = \frac{ {\textstyle \sum_{i}}N_i }{\left \| {\textstyle \sum_{i}}N_i \right \| }Nv​=∥∥​∑i​Ni​∥∥​∑i​Ni​​ )
  • 每个顶点着色一次，每个顶点都有了颜色。
  • 中间的颜色利用重心插值计算。

3.3. Phong Shading

为每个像素着色：

  • 求出每个像素的法向量(通过给顶点法线做插值的方法)
  • 对每个像素进行一次着色

4. 纹理映射

一个物体公用一个着色器模型，只是漫反射系数发生了变化。我们希望在物体的不同位置定义不同属性，引入纹理映射。纹理的作用是定义任何一个点的基本属性。任何3D物体的纹理都是2D的。
纹理坐标通常使用 uv 表示，不管纹理的长宽比是多少，我们都默认它是 [0, 1]。

5. Interpolation(插值) Across Triangle

为什么要做插值？

  • 指定顶点上的值。
  • 在三角形内部有一个平滑的过度。
  • 纹理坐标、颜色、法向量、…… 都可以做插值

7.1 纹理太小怎么办？

纹理分辨率不够会出现锯齿:

纹理分辨率不够的情况下，我们任然希望得到第二个或者第三个图片中的效果。

双线性插值（Bilinear Interpolation）：

  • 黑色的点表示像素的中心。
  • 我们期望在红色点进行采样。
  • 取 4 个最近的纹理，并标记上对应的纹理值。(s,t) 分别是水平和垂直的偏移量
  • 设计一个线性插值函数

lerp(x,v0,v1)=v0+x(v1−v0)lerp(x, v_0, v_1) = v_0 + x(v_1 - v_0) lerp(x,v0​,v1​)=v0​+x(v1​−v0​)

  • 做两次水平的线性插值

v0=lerp(s,v00,v10)v1=lerp(s,v01,v11)\begin{aligned} v_0 &= lerp(s, v_{00}, v_{10}) \\ v_1 &= lerp(s, v_{01}, v_{11}) \end{aligned} v0​v1​​=lerp(s,v00​,v10​)=lerp(s,v01​,v11​)​

  • 最后做一次垂直的插值
            f(x,y)=lerp(t,v0,v1)f(x, y) = lerp(t, v_0, v_1)f(x,y)=lerp(t,v0​,v1​)

Bicubic：
在数学中，双三次插值（Bicubic Interpolation）是对一组数据进行三次插值的扩展，用于在二维规则网格上插值数据点1。与双线性插值或最近邻插值得到的表面相比，插值表面（指内核形状，而非图像）更为平滑1。

双三次插值可以使用拉格朗日多项式、三次样条或三次卷积算法来完成。在图像处理中，当速度不是问题时，双三次插值通常被选为图像重采样的方法，优于双线性或最近邻插值1。与只考虑4个像素（2×2）的双线性插值不同，双三次插值考虑16个像素（4×4）

7.2 纹理太大怎么办？

纹理太大反而会出现另外的问题，我们需要做范围查询。

锯齿和摩尔纹我们通常可以使用超采样的方式来反走样，但这也意味着更高的消耗。

MipMap：
Mipmap 是一系列的纹理图像，它们是原始纹理的缩小版本。每个级别的 mipmap 都是前一个级别的一半大小。这个系列从原始纹理开始，然后逐渐缩小，直到最后一个 mipmap 只有一个像素大小。当一个物体在屏幕上变小或远离观察者时，mipmap 技术会选择合适的纹理级别进行渲染。这样可以减少渲染时间，并且可以减少因纹理缩小而产生的混叠效应。

计算mipmap的层：

将像素坐标映射到纹理坐标，利用相邻纹理坐标之间的关系求出一个 texel 并的关系：

L=max((dudx)2+(dvdx)2,(dudy)2+(dvdy)2)L = max(\sqrt{\left ( \frac{du}{dx} \right )^2 + \left ( \frac{dv}{dx} \right )^2 } , \sqrt{\left ( \frac{du}{dy} \right )^2 + \left ( \frac{dv}{dy} \right )^2 } ) L=max((dxdu​)2+(dxdv​)2​,(dydu​)2+(dydv​)2​)

D=log2LD=log_2L D=log2​L

有的时候 D = 1.8 我们是取第二层还是第一层呢？感觉都不是很好。我们可以进一步做插值。在 D & D+1 层都做线性插值，再在 D & D+1 之间在进行插值。

7.3. Mipmap 的局限

Mipmap 只能做正方型的映射所以导致在映射一些长条形的纹理时会出现比较模糊的现象。

解决方案：
我们可以使用各向异性过滤(Anisotropic Filtering)

它的弊端是需要在原来的基础上再增加三倍的内存。而mipmap只需要在原来的基础上增加 1/3。
然而各向异性过滤也不能解决所有的问题，如上图 texture space 中 右上角的长条形纹理，任然需要框选很大一部分。

还有一种办法：EWA filtering，它可以处理一些不规则的纹理。

8. Environment Map

什么是Environment Map？假设我们站在一个房间里面，我们可以往四面八方看，大家会发现，四面八方都可以看到物体，所有四面八方都会来自有光，任何一个方向，不管是直接光照还是间接光照。如果我们可以把任何一个方向的光都记录下来，这就是环境光照或环境贴图。

9. Texture can affect shading

纹理并不是仅仅表示颜色，它可以定义任何不同位置，任何不同的属性。

如果我们存储的是 height/normal 会怎么样？应用 bump/normal 映射，可以伪造一些几何细节。

9.1 凹凸贴图(bump mapping)

可以添加更多的表面细节但不需要增加更多的三角形。通过定义不同位置的高度，把表面每个像素的法线都做一个扰动。也就是，利用临近位置的高度差，从新计算它的法线。

黑线表示原本光滑的物体表面。我们应用一个凹凸贴图，它告诉了我们相对的高度变化，例如任意一点p，黄色的线表示它的相对高度，通过相对高度制作出一个虚假的法线。

如何取扰动法线呢？

  • 我们假设原平面的法线为 n§ = (0,1)
  • 求出任意一个点 p 的切线 dp = c * [h(p+1) - h§]，c 是引入的一个系数用于控制凹凸贴图对法线的影响。
  • 切线编导法线 n§ = (-dp, 1).normalized()

在 3d 的情况下如何求法线?

  • 假设原平面法线 n§ = (0, 0, 1)
  • p 点的倒数：
    • dp/du = c1 * [h(u + 1) - h(u)]
    • dp/dv = c2 * [h(v + 1) - h(v)]
  • n = (-dp/du, -dp/dv, 1).normalized()

note: 这是一个局部的坐标系

9.2 位移贴图(displacement mapping)

比起凹凸贴图，它实际移动了顶点。

代价：要求模型的三角形要足够的细，要求模型能完整的反应纹理的高度变化。

我并不需要一开始就要求模型足够的细致，我们可以按需求去做一些曲面细分（DirectX 支持动态的曲面细分）。
